# Todo Color Guessing Game

A small web game that guesses colors — simple HTML/CSS/JS project.

Files:
- `index.html` — main page
- `style.css` — styles
- `script.js` — game logic

How to run locally:
1. Open `index.html` in your browser, or run a local static server such as:

```powershell
# using Python 3
Set-Location -Path 'C:\Users\Umar\Desktop\New folder'
python -m http.server 8000
# then open http://localhost:8000
```
